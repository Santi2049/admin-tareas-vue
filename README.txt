Instrucciones para ejecutar la aplicacion: 
1.- Instalar el servidor de json-server en cmd del sistema con el siguiente comando:

npm install json-server 

2.- Clonar el repositorio mediante el link de github 

3.- Abrir el proyecto en visual code y abrir una nueva terminal y ejecutar el siguiente comando :


npm install 

4.- Esto instalara las dependencias necesarias para ejecutar el proyecto, finalmente en la terminal ejecutar el siguiente comando: 

npm run start
